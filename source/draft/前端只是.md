#### 如何理解闭包
js的作用域为链式作用域，子级能够读到各上级的变量，父级无法读到子级的变量，而通过闭包的方式，可以在函数外层访问到函数内部的变量，例子

```javascript
　function f1(){
　　　　var n=999;
　　　　function f2(){
　　　　　　alert(n); 
　　　　}
　　　　return f2;
　　}
　　var result=f1();
　　result(); // 999
```
